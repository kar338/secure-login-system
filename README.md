# Secure Login System

A secure Flask-based login system implementing password hashing, session management, and protection against SQL injection.

## Features

- User Registration
- User Login
- bcrypt Password Hashing
- SQLite Database
- Session Management
- Logout Functionality
- Input Validation
- SQL Injection Protection

## Technologies

- Python
- Flask
- SQLite
- bcrypt

## Installation

```bash
pip install flask bcrypt
```

## Run

```bash
python app.py
```

Open:

http://127.0.0.1:5000

## Security Controls

- Passwords stored as bcrypt hashes
- Parameterized SQL queries
- Session-based authentication
- Input validation
- Protected dashboard access
